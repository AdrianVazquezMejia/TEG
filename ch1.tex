\section{Fundamentos de las redes WiFi malladas}


Una red WiFi mallada es una red inalámbrica establecida en la banda establecidas para comunicación WiFi (sea 2,4 GHz o 5GHz), donde hay estaciones que soportan comunicación de multiples saltos (multi-hop) para transferir información en la red. Así mismo se tiene que, el enrutamiento y la capacidades de reenvios de datos residen en la capa de Control de Acceso Medio (MAC) \cite{article1}.
