Se recomienda implementar un enrutamiento dinámico en la red, para observar los beneficios respecto a el enrutmiento estático. 

Se propone aplicar una rutina de búsqueda de esclavos en cada nodo, así cada nodo detectaría los esclavos que posee conectados, para disminuir el tiempo de configuración de la red.

Diseñar una interfaz gráfica para configurar la red,  para así disminuir el tiempo y los errores  de establecimiento de la red.

Se propone implementar la configuración de canal \wf  y la llaves locales mediante \mb, para poseer mayor extensibilidad de red.

Es recomendable  extender la configuración para el soporte de más opciones de \mb serial, como el modo ANSII.

En el hardware del nodo, se debería agregar la interfaz mecánica de RJ-45 muy popular en la industria. Así como puertos de entrada salida para otras posibles funciones de monitoreo.

Implementar el modo de maestro \mb en los nodos para que exista la posibilidad de que el nodo maestro funcione como puerta de enlace a otras redes, por ejemplo \mb TCP/IP.
